le koj zaj lug tuabneeg (tsi yog lugnruag)
teebmeem taamsim ntawd yog tug nyaab thaam hluas nraug.  leej tub
ntxhuvsab es xaav paub txug saib yog ua lecaag.  tug nyaab tsi xaav
qha vim kev txaajmuag (ua txhum) es nwg txhaj le quaj hab hu pusliv
tuaj dlaws teebmeem.
tug pusliv laug noj ntau dlavmov ces nwg paub txug noobneej txujkev
uaneej vim nwg paub qhov yog hab qhob tsi yog.  tamsis qhov ntawd
dlaws tsi tau tsevneeg ntawd qhov teebmeem hab ua nwg qhov num tsi
tav.
cov pusliv hluas noj tsawg dlavmov ces puab paub tsawg txug noobneej
txujkev uaneej.  puab pum qhov kws tim qhovtwg dlawm qhovntawd rua
taamsim ntawd xwb.  puab qhov num ces yog le ntawd.
tug nyaab yuav nti kuas dlim ces yuav tau has yaam kws nwg yuav ntsib
losyog xaav has tas nwg tug quasyawg yuav muab nwg tua.  has txug qhov
tua ces tub ceevxwm yuav tsum sawm lug tswj xwb vim qhov ntawd dlu
puab lub sab txug puab teg dlejnum lawm.
tub ceebxwm nteg tug quasyawg vim puab xaav rhuavtsheem tug tuabneeg
kws yuav txuv lwm tug txujsa.  yog puab tsi dlawm, nwg zwm rua huv
puab lub sab txug puab teg dlejnum lawm.  puab yuav tsum nteg tug
quasyawg xwb.
leej txivyawm pum qhov xwmtxheej ntawd yaam qaugzug kawg vim nwg ua
tsi tau lecaag rua lub sibhawm ntawd.  qhov kawg nwg txhaj xaiv
txujkev moog dlu tuabneeg lubsab (lub chaw paub tugkheej, kev phem,
kev zoo, paub vaajtswv, hab paub dlaab).
tug pusliv laug zooj hab pum qhovntawd, nwg sab tsi kheev nyobtsaam
cov pusliv coj tsi ncaaj lawm, nwg txhaj le noog tug mivnyuas.  thaus
nwg paub txug lawm has tas, xwmtxheej sau yaajceeb yog le tuabneeg lub
qhovmuag pum tamsis tejnum ntawd zoo le tsi ncaajnceeg.  qhov
ncaajnceeg ces yog nyob rua huv yeebceeb (nruabsab kws muaj kev
ncaajnceeg, paub qhov phem qhov zoo, paub vaajtswv hab paub dlaab).

nyaab sau nwg tej txajtsim dlaag nrug nwg moog lawm.
