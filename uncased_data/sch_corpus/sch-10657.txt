here is what i usually do to my computer(s):
1.  use anti-virus program (i.e. macafee, norton, etc.) scan the
entire hdd for viruses.  yog muaj virus yuav tsum muab tshem.  tshem
tsis tau yuav tsum reformat hdd es pib dua tshiab koj tej nyuam nyiaj
kub (refering to your files) thiaj li tsis piam.
2.  yog tsis muaj viruses, mus rau hauv control panel, add/delete
programs.  go through cov programs uas automatically install los yog
tej yam programs uas koj tsis siv lawm - un-install kom tag.
3.  mus rau hauv internet explorer.  tools -> internet options.  click
delete...  thaum tawm qhov windows tshiab tuaj, click qhov uas hais
tias delete all.  qhov delete tag nrho ib puas tsav yam files uas ie
khaws tseg rau hauv koj lub computer under koj tus login/password.
4.  mus rau system tools thiab siv disk cleanup program.  delete kom
tag ib puas tsav yam files.
5.  mus rau system tools thiab siv disk defragmenter program.  qhov no
yuav muab koj cov files coj los tso ua ke physically thiag logically
rau hauv koj lub hdd.  yog koj tsis tau ua dua li no li mas qhov step
4 thiab 5 no tej zaum yuav siv tsheej hnub tej.
sim li no tag yog tias tseem nkag li vaub kib xwb no ces mus hauv task
manager seb muaj pes tsawg qhov "tasks" running.  average mas yog
30-40 los yog ntau mas 50 tej xwb nawj.  tshaj ntawv lawm ces for sure
koj lub computer yuav khiav qeeb.  saib seb qhov task twg uas koj tsis
recognize.  yog koj tsis paub no ces mus tshawb hauv internet los yog
microsoft lub website.   tej thaum yog tasks uas yuav tsum muaj, tej
thaum yog spyware programs xwb.
ultimately, yog ua li cas tsis tau li no los yeej yuav tsum yog
reformat xwb.
thaum koj reformat, nco ntsoov backup koj cov files uas koj need nawj
tsis li pab tsis tau.
hopefully, this helps.
