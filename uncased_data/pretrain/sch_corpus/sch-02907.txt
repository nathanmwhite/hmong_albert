2.	cov thawjcoj tseem nyob theem siv “tsuv zug” kws tsi yog thawjcoj.

puab tsuas tswj kuas puab tau xwb vim puab tsi muaj kev ncaajnceeg.

puab tsi paub txug puab tugkheej.
puab tsuas pum qhov lwm tug pum puab xwb, ces puab yuav tsum ua kuas hum le lwm tug nyam.
qhovntawd yog qhov kws yuamkev ib tam dlhau moog rua ib tam.

3.	moob pheej tsiv yog moob dlaig rua them “kaajsab” kws tsi paub txhawj le.
thaus lub neej hab txujsa nyuab ces ncutxug lub qub neej (hmong kingdom) vim tsi paub yog rua lub neej hab txujsa.
xaav npaum twg los tsi pum tom hauvntej le vim cov thawjcoj coj tsi ncaaj, huas cov saub los coj lub neej vwm quasntsuav le hab.
thaus ntawd ces moob txhajle nyob rua lub neej npausuav kuas muaj tug cawmsheej (tsuv zug + saub).
qhov tseeb tag, moob tsi paub moob tugkheej.
